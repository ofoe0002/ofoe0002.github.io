Notes Page

  • The Anchor tag: ('a')This is used to create an anchor for hyperlinks
  • The Paragraph tag: ('p')This tag is used to define the beginning of a paragraph within the body
  • The List tag: ('li')This is used to define an item in a list
  • The Body tag: ('body')This tag defines the content that will be displayed within the html document
  • The Line break tag: ('br')This tag is used to define a break in a line of text.
